# v0.1.0

First public release of the WeChat DevTools automation skill.

## Includes

- reusable Codex skill for WeChat Mini Program DevTools workflows
- project-local bootstrap script for automation scaffolding
- validation script for generated bootstrap output
- installer script for quick local skill setup
- GitHub templates for public maintenance

## Notes

- designed for macOS-based WeChat DevTools workflows
- target project must already be a valid Mini Program project
- project-specific diagnostics pages and cloud auth still need to be implemented in each app
